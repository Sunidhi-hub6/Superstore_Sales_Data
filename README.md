# Superstore_Sales_Data
Task 2: Data Visualization and Storytelling - 
Superstore Dataset

Objective:
Create visualizations that convey a compelling story.

Dataset Used:
Superstore_Sales_Profit (Kaggle)

Tools Used:
- Power BI Desktop → Modeling, DAX, visuals
- Excel/CSV → Measure definitions, storyboard planning
- GitHub

Problem:
Superstore faces uneven profitability despite solid sales growth. Discounts appear to be eroding margins across categories, regions, and sub-categories, creating a need for clear business insights to guide pricing and inventory decisions.

Steps Performed:
- Data cleaning → Removed nulls, standardized dates, calculated margin % field
- DAX measures → Created YoY growth, profit margin, discount impact KPIs
- Visual storytelling → 5-page dashboard sequence (Executive Overview → Category Performance → Discount Impact → Regional Insights → Profit and Discount)
- Design principles → Right chart per question, limited colors, direct annotations, business context over decoration

Output Files Included:
- Original dataset
- Visualised Data (Power BI)
- Screenshot

Result:
The dataset is now visualized, and ready for further analysis of Power BI report template that any analyst can open and immediately understand Superstore's core business story: strong sales growth masked by discount-driven margin pressure.
